typedef void(* hsm_callback_popped_context)(hsm_context_t *)

Hear about your context object after just after its been popped.

Can be used to free memory for state instance data you allocated during state entry. It's designed like a destructor and so doesn't provide any parameters other than itself.

Parameters:
hsm_context_t*The context being freed

hsm_context HsmContextAlloc ( size_t  size)

Helper to create a new context object.

It will be auto-freed when popped.

Parameters:
sizeTotal size of the user context structure.
Returns:
Newly allocated context data, NULL if the allocation failed.

Sets the hsm_context_rec::popped handler to an internal free function. You can also allocate your objects on your own if you want more control.
