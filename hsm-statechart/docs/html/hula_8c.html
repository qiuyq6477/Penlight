hsm_bool HulaMatchEvent ( const char *  spec,
const char *  test 
)

Match an event specification to a triggered event.

The matching takes into account hierarchical events. This behavior is the default behavior of lua statecharts, user code can customize this behavior via hula_callback_is_event.

Parameters:
specEvent name specified in the lua defined statechart.
testEvent name signaled by lua.
Returns:
HSM_TRUE if they match
See also:
hula_callback_is_event

event.item <- specified event(s) we want to handle event.item.click <- a triggered event we match event.item <- another match event.mouse.drag <- no match event.items <- no match event <- no match
